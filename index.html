📋 Requisitos para Unirse 📋

  • Tener Euro Truck Simulator 2 (ETS2)
  • Tener el DLC de buses (Bus Connector)
  • Unirse a nuestro servidor de Discord
  • Tener micrófono para comunicación (opcional)
  • Respetar las normas de conducción
  • Ser mayor de 16 años o tener autorización
  • Completar el período de prueba
  • Participar en al menos 2 convoyes por mes
